# BlogAggregator

A program which collects RSS feeds and stores them in a database

## Requirements

- golang
- postgres

  ## Usage

  clone the project and run `go install`

  ## Available commands:
  
- go run . login <user>
- go run . register <user>
- go run . reset
- go run . users
- go run . addfeed <name> <url>
- go run . feeds
- go run . fullow
- go run . following
- go run . agg
- go run . browse
